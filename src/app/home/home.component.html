User Generator API

An API for generating random user data. Like Lorem Ipsum, but for people. 

  {{'{'}}
    id: 2
    name: "Rachel"
    surname: "Scott"
    email: "rachel.scott73@example.com"
    date_of_birth: 14634
  {{'}'}}

When to use:

UserApi is an online REST API that you can use whenever you need some fake data. It can be in a README on GitHub, for a demo on CodeSandbox, in code examples on Stack Overflow, ...or simply to test things locally.
